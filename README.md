## Structure du Projet

```bash
src/
├── api/             # Configuration Axios et Intercepteurs
├── components/      # Composants UI (Form, History, Footer)
├── services/        # Logique API (Analysis, History Services)
├── hooks/           # Hooks personnalisés pour la data
├── types/           # Interfaces TypeScript (HistoryItem, etc.)
└── App.tsx          # Point d'entrée de l'application

## Installation

### 1. Cloner le projet
```bash
git clone  
cd  
```

### 2. Configuration

```bash
# Installer les dépendances
npm install

# Lancer l'application en mode développement
npm run dev

```
